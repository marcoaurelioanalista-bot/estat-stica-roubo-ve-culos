# Análise Estatística de Roubo de Veículos 🚗📊

Este projeto foi desenvolvido com foco em Inteligência de Mercado e Análise de Dados aplicadas ao setor de Seguros e Segurança Pública. O objetivo principal é estruturar dados brutos de sinistros para identificar padrões comportamentais, geográficos e temporais que auxiliem na precificação de riscos e tomada de decisões estratégicas.

## 🎯 Contexto de Negócio
Com base em uma vivência profissional de 19 anos no mercado de seguros (atuando em grandes instituições como MAPFRE e Banco Santander), este repositório unifica a expertise em regras de negócio de subscrição de riscos com técnicas modernas de manipulação e análise de dados.

## 🛠️ Tecnologias e Conceitos Aplicados
* **Lógica de Programação e Dados** para estruturação do problema.
* **Manipulação de Arquivos e Dados Estatísticos**.
* **Versionamento de Código** utilizando Git e GitHub.
* **Modelagem Conceitual** voltada para análise de sinistros e enquadramento de veículos.

## 🚀 Próximos Passos do Projeto
* Implementação de scripts em Python para automatização da limpeza de dados brutos (ETL).
* Criação de consultas avançadas em SQL para cruzamento de dados de frotas e ocorrências.
* Construção de um Dashboard interativo em Power BI focado nos principais KPIs de sinistralidade por região.
